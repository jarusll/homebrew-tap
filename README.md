# Jarusll Tap

## How do I install these formulae?

`brew install jarusll/tap/<formula>`

Or `brew tap jarusll/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jarusll/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
